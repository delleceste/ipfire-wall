How to specify a list of ip addresses

To insert a list of IP addresses proceed like this:

  • type the first IP address;
  • click on the last apply button on the right. The combo box drop down list should now show the address;
  • click on the + button to add a second element;
  • type in the second ip address in the white space;
  • click once again on the apply button on the right.
  • Proceed again in the same way until done.
To remove an IP address from the list:
  • from the drop down list of entries, select the one you want to remove;
  • click on the - button.
