Alternative Network Address Translation ruleset

iqfire-wall allows the root user to specify a NAT policy file different from the default used.

Should you need to test temporarily another set of translation rules, specify here the position on the file system and the name of the file you want to use.

This setting needs the firewall be restarted to be applied.

Restore the default permission/denial rulesets clicking on the appropriate button Restore Default Paths below.

This setting is available only for the root user.
