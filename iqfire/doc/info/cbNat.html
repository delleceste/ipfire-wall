Network Address Translation

This option enables Network Address Translation globally. Network Address Translation (NAT) is a feature that allows changing the source or destination address (or source or destination port) of a packet to redirect it to another destination (or service, for port translation) or to make it appear as coming from another host (for source NAT).

For instance, with Network Address Translation you are able to redirect all the traffic from host A to host B towards host C. Moreover, it is possible to redirect a connection to port x towards another port number.

A particular case of NAT is represented by masquerade: used particularly in gateways, it allows to share an internet connection among multiple hosts referring to the gateway connected to the Internet.

Configuring NAT requires administrator privileges: this means that only root can add/remove NAT rules and enable/disable this option.
