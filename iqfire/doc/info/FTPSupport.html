Passive FTP support

This check box enables the passive FTP support for this rule. It is normally applied to a permission rule allowing the ftp control flow (i.e. towards the port 21/ftp).
For each ftp control connection, a new table is created in the kernel space firewall to allow the related connection to take place successfully.

The payload data is inspected to look for the ftp data connection parameters negotiated in the control session. This procedure is necessary to allow the FTP transfers.
