Ignore the selected packets

Ignoring the packets means adding it to the list of the Connections waiting for authorization. You can later revise the list and Accept or block its elements.

This is the default behaviour when you directly click the Ignore All push button. In that case, all the elements are added to the list of the Connections waiting for authorization. This brings two main advantages:

  1. you can decide later on, in a moment's peace, the lot of the packets;
  2. you avoid that the same packets continue being enqueued to the popup.

Remember that to the ignored packets the default policy of the firewall is applied. This means that no rule is inserted into the firewall's policies for the ignored packets.
