Pending packets

Here iqFirewall lists all the packets arrived at your linux box which did not match a rule. Normally, when this event happens, a popup message should appear in the proximity of the system tray icon in your desktop panel. See also the note below...

Here you have a list of options for this window:

  • Double click on a column of any item and change its contents if you like, or
  • right click on a column to get a popup with some default values for the content of that column;
  • click on one of the buttons below to add the rule to the blocked list, to the permission list or simply to remove the item from the list.

The fields represented in the columns of this window have the same meaning that they assume in the rules tree window, to which we send the reader back.

Note: if this tree is empty or you suspect that the feature is disabled, right click on the system tray icon and choose popups... and tick Unknown Connections.
