Vary the length of the blocked packets needle

Click the arrows or type a value from 0.0 to 1.0 to vary the relative length of the red needle in the dashboard represented in the System Tray Icon.

A value of 0.0 means no red needle, a value of 1.0 draws a needle long the full available length of the radius of the System Tray Icon.

There is no need to click on the Apply button below to save the modifications and apply them.
