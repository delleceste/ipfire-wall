Add a new Style sheet

Click on the Button to browse the file system and load a new css stylesheet written to personalize the user interface.

The new stylesheet will be added to the table above and to apply it, you must select the corresponding row in the table itself. Remember that, to be able to select rows on the table, you must first uncheck the box Qt Default Style.

Finally, click on the Apply button to load the style, or click on Cancel to discard modifications.

If you are unhappy with your new style, then you can also remove it from the list on the table by clicking on the Remove button, after selecting the row in the table corresponding to the document you want to remove from the list (not from the file system, of course).
