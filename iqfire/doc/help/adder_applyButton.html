Apply the new rule or the modifications to an existing rule

Click here to insert a new rule or apply the modifications you have made to an existing rule.

Observation

  • Remember that this option inserts the rule into the tree view only! This means that you will have to
    • Apply it pressing also the Apply button below the tree widget (and this will temporarily load it into the kernel);
    • if happy with the changes, save it permanently clicking the Save button in the tree view, next to the Apply button mentioned at the previous point. Actually, the Save button writes on the disk the new or modified rule, to make it available at next iqfirewall startup or, if you are administrator, at next system boot.
