Alternative Documentation Help File

iqfire-wall allows each user to specify documentation files different from the default used.

Should you need to view another set of documentation files specify here the position on the file system of the folder containing all the html and css documents.
This feature allows you to improve the existent documentation and try it on your working directory!

This setting does not need the firewall be restarted to be applied.

Beware that the file names must be the same of those contained in the default directory.

Restore the default positions clicking on the appropriate button Restore Default Paths below.
