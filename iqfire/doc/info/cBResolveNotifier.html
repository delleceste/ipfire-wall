Resolve IP addresses and ports in the popup items

A new item in the popup displays the following information:

  • packet direction (input and output supported);
  • protocol;
  • source and destination IP addresses;
  • source and destination port numbers;
  • name of the network interface through which the packet passed;
  • suggested rule name;
  • possibility to notify when a network packet matches the item.

The IP addresses and ports are numbers, but if you enable their resolution, then they will be shown with their associated names together with the numeric form.

The address and port names are available as tooltips when the mouse stops over an item of the popup, or in this informational panel, always when the mouse moves over the items or selects one of them.
