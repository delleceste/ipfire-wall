Natural language editor

In this page iqfire wall presents its natural language editor. It is a very simple editor where you can type the text aimed at expressing the set of rules you desire in a natural language form.

If you are in doubt about the syntax or the appropriate words that can be used in natural language, you can have a look at the following resources:

  • verbs to express firewall actions;
  • dictionary of keywords used by the firewall;
  • grammar: a set of grammatical rules one must keep in mind when using special keywords.

While reading the resources above, consider resizing this help panel and in case hiding the info panel above, to optimize space. This help panel can be resized with the mouse by positioning the pointer over the top and left sizes.
