Enable NAT globally

Check this box if you are going to use the Network Address Translation (NAT) functionality of iqfire-wall. If there are no NAT rules and you have no plans to make the software act as a NAT firewall, then disabling this is a good choice, since it speeds up computing.

If this feature is disabled, then all the NAT checks are disabled and the firewall works only as a packet filter.

If NAT rules are loaded, but this check box is not enabled, then those rules will not be considered.
