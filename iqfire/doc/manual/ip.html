IP addresses

An IP address is a unique identifier used by computers to communicate with each other on a computer network, using the internet protocol (IP) standard. Some addresses are intended to be unique in the whole Internet, other are unique in a more restricted context, such as a home or an enterprise, or a university (local or private addresses).

Representation

An IP address is represented by four decimal numbers (octets) separated by a dot. Each decimal number ranges from 0 to 255.
Examples are 192.168.0.1 or 147.131.40.11.

Classes of addresses

All the possible IP addresses are grouped in classes of networks:

  • class A: the first octet ranges from 1 to 126;
  • class B: the first octet ranges from 128 to 191;
  • class C: the first octet ranges from 192 to 223;
  • The class D is not currently used.
    Special meanings of some addresses:
  • The special address starting from 127 represents the local computer, independently from the real network, and it is mainly used for testing purposes;
  • The special addresses starting from 224 are reserved for particular uses, such as multicasting.

    Machines which are not connected to the outside world need not to have globally unique IP addresses. The RFC 1918 defines the ranges of addresses for private networks. These addresses are never routed and belong to the following intervals:

    • 10.0.0.0 - 10.255.255.255, for class A;
    • 172.16.0.0 - 172.31.255.255, for class B;
    • 192.168.0.0 - 192.168.255.255, for class C;
