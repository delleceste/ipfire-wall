Resolve IP addresses and port numbers

This option offers the possibility to display on the popup that notifies a rule being applied to a connection the name representation of the IP addresses and ports.

Actually, IP addresses and ports are numbers, but this option for each of them starts a sub process that tries to resolve the numbers into names.

For instance, the port number 22 is associated to the ssh service, and the IP address 62.149.140.40 to the name www.giacomos.it.
