Filter on the interface name

Click on the combo box and decide to:

  • not to filter the packet on the network interface name (ANY). This means that the filter rule will not consider at all the interface name;
  • filter on the basis of the interface name, be it input or output: with this option it is enough that the name provided matches any between the input interface name or the output interface name(INPUT OR OUTPUT);
  • filter on the basis of the input interface name (INPUT);
  • filter on the basis of the output interface name (OUTPUT);
  • filter on the basis of both the output interface name and input interface name (INPUT AND OUTPUT). In this case, two boxes will appear on the right of this combo box to let you fill in the two names you want to filter on.
