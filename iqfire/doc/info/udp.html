The UDP protocol

The UDP protocol is the acronym for User Datagram Protocol and it is not connection-oriented (connectionless) and unreliable.

    The UDP protocol is also used in many Internet services; among these:
  • the Domain Name System (DNS), used to resolve the internet names, such as www.google.com, into the numeric address of a machine in the network;
  • audio/video streaming protocols;
  • the voice over IP protocol (VoIP)...
    • The connectionless and unreliable characteristics of the User Datagram Protocol make it suitable for faster and more efficient communications. Time-sensitive applications often adopt UDP because dropped packets are preferable to delayed packets (audio video streaming for instance).
