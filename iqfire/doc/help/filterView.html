Filter view

Move your mouse over the elements of this widget to get specific help. Once you have setup the desired fields, click on the check box Apply the filter view and go back to the console window (Menu -> Window -> Console) or click on the Back arrow in the tool bar.

Important note

The settings in this window do not affect the firewall packet filter or NAT activity.
They only make a selection of which packets to show or not in the console, to allow the user to concentrate only in a subset of all the packets processed.

For instance, i could be interested in watching on the console only the TCP packets, or only the packets initiating successfully a connection.
