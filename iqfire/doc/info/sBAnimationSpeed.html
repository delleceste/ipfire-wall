Refresh speed of the system tray icon

The system tray icon of iqfire-wall indicates the status of the firewall, when the animation is enabled.

In particular, every Refresh speed of the animation milliseconds, two rectangles are drawn over the icon. One is green, and represents the allowed packets filtered in that time interval, and the other is red, representing the blocked packets. Each rectangle has the height proportional to the number of filtered packets in the specified interval.

The user, observing the system tray activity, has an idea of the state of the firewall at each instant.

Of course, drawing very often over the icon requires a bit of cpu calculations and this might not be efficient for older systems: tune the animation speed to obtain a good compromise between a good looking animation and a good performance of your system.

A good value for a recent system could be around 800 milliseconds, older systems might like an interval of one second or more.
