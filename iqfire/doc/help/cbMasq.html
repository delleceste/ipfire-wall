Enable Masquerade globally

Check this box if you are going to use the masquerade functionality of iqfire-wall. If there are no masquerade rules and you have no plans to make the firewall act as a NAT/masquerade firewall, then disabling this is a good choice, since it speeds up computing.

If this feature is disabled, then all the masquerade checks are disabled.

If masquerade rules are loaded, but this check box is not ticked, then those rules will not be considered.
