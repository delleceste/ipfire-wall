Create a permission rule from the selected items

  • select an item from the list with the left button of the mouse;
  • double click on an element to change its contents (optional);
  • click on this radio button and then apply clicking on the Apply push button on the right bottom corner of the popup.

    • This creates a new rule, and the rule is enqueued to the list of the permission rules already present in your firewall.

    You can see the rules loaded into the firewall in the rule tree section.
