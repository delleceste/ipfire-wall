Packets without a permission or denial rule

In this popup window iqfire-wall reports all the packets arrived to the firewall that do not satisfy any rule: not a permission one nor a denial one.

For this reason, no explicit decision could have been taken and, in cases like these, the default policy is applied. The default policy is decided by the administrator of the computer in the settings configuration window of iqfire-wall.

Some different actions can be taken here:

  • close the window with the usual x button on the title: this ignores all the information gathered;
  • click on the Ignore All button, that directly adds all the items to the list of the already notified items list and avoid that the same information is shown again when another identical packet arrives to the firewall;
  • select one or more items and choose one of the options provided by the four radio buttons below:
    • Ask again: ignore the information contained in the selected item(s);
    • Accept: transform the item into a permission rule. This takes all the fields in the item and builds a rule starting from them. Before adding a rule you can edit the items to change some values, like adding a netmask of an internet address or setting to any some field (right click on a column and click on the Any... menu item);
    • Block: transform the item into a denial rule. This works as the case above, but blocks the packets that match the new rule;
    • Ignore: add the item to the list of the already notified items list and avoid that the same information is shown again when another identical packet arrives to the firewall;

Notes

You can select all the items by checking the checkbox All. You can deselect all the items unchecking the All checkbox.

Double click on any column you like to change its contents. For instance, you could desire to substitute the address 192.168.0.2 to an interval 192.168.0.0-192.168.0.10 or 192.168.0.0/255.255.255.0, or set to any one of the fields you are not interested in.

Notice that, when no items are selected (i.e. when the popup first appears) the radio buttons at the bottom of the window are disabled. You have to select one or more items to enable them. When you select one or more items the text in the Ignore All button changes into Apply. Actually, the meaning of the button changes and Apply indicates that you must press that button if you want to apply the option chosen in one of the radio buttons.

Moving the mouse through the items will provide a packet summary in html format which is displayed in the informational panel above.

Selecting or clicking on an item will display once the summary in the informational panel above and won't change anymore with the mouse moves over the other items.
