Create a denial rule from the selected item(s)

It is possible to add the selected items in the list of the denial rules. The items will be removed from this list and for each of them a new rule will be added to the denial rule tree.

Before adding a rule inspired to the item, you can double click on each column if you like to change some values, like for example to transform an IP address into an interval, or add a netmask, or set to any some fields you are not interested in.

To set to any some field, just right click over the corresponding column and a popup menu will help you do the task.
