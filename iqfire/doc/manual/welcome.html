iqFirewall is a simple graphical user interface which helps you protect your personal computer from the outside internet world, while providing amusing tools to configure the rules of the firewall and to observe the statistics of the network traffic filtering.

This page provides access to the main functionalities of the firewall, as well as its online manual.
iqFirewall consists of a set of tools to configure and and control the firewall actions. Some of them are:

  • A monitoring console, showing the traffic being processed, in full analogy with the former ipfirewall;
  • a rule tree window, representing the characteristics and the order of the rules being applied by the firewall. There you can edit and add or remove your personal rules, also if you are not the system administrator! For instance, currently $accRulesNum$ permission rules are loaded and $denRulesNum$ denial ones;
  • a statistics window, showing the percentage of the blocked/accepted packets over time! Until now, $allowed$ network packets nave been allowed, while $blocked$ blocked.
  • An innovative linux routing block scheme window, allowing the user to add new rules and to observe per direction statistics.

If you are curious you can directly access one of the above functionalities, or navigate through the menus and the tool bar to discover other aspects of this Linux firewall, such as the right navigation panel or the information and help right panels, suitable to give you hints about all the elements of the firewall interface!
Moreover, if you are new to linux firewalling or iqFirewall, start reading the integrated manual.
Read also the iqfire-wall online manual to obtain the most up to date information about using and configuring the software.
To get more in depth documentation about the implementation and the usage of the ipfire-wall framework, visit the ipfire-wall web page. Finally, the ipfire-wall scenarios page shows some contexts where the firewall can be installed and configured.
