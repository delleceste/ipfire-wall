The TCP protocol

The TCP protocol is the acronym for Transmission Control Protocol and it is connection-oriented and reliable.

    The TCP protocol is used in many Internet services:
  • the world wide web (www);
  • the electronic mail;
  • the secure services over the web (http-secure)
  • various reliable and connection oriented protocols, as ftp, ssh and telnet.
Reliable and connection oriented characteristics refer to the fact that TCP guarantees that all packets arrive at destination in the same order they are sent by the source.
