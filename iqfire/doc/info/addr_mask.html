Address Mask

Computer networks that use the internet protocol use the concept of subnetwork to divide the address space into ranges of logical addresses. A subnet is identified by its base address and the subnet mask.

Specifying here an address and netmask pair allows to declare an interval of addresses to which the rule must be applied.
In fact, iqfirewall converts a couple IP/ netmask into an interval of IPs.

For instance:

  • 192.168.0.0/255.255.255.0 is equivalent to 192.168.0.0-192.168.0.255, and
  • 192.168.0.0/255.255.255.248 is equivalent to 192.168.0.0-192.168.0.7

If you need to learn more about subnetworks and netmasks you can read the IP addresses manual page, which contains external web links treating such argument.
