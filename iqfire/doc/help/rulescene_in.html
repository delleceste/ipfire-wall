The Input block

  • Right click on the rectangle to add a permission or a denial rule on the input stage;
  • Single click on the item to view the statistics of the blocked packets for the input stage.

Note

  • If you click to view the statistics, to return to this help window you must go to the sidebar contents toolbar and hide and show again the help panel (click two times on the yellow bulb icon in the toolbar).
  • To view the complete information about the statistics, go to the statistics page.
