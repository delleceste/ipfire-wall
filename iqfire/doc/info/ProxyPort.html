The proxy port number

To retrieve the news about the software, you must access the world wide web service. This means you have to be connected to the Internet. Moreover, if you normally connect to the www passing through a http proxy, you have to configure here the same parameters you configured to navigate the web in your web browser.

Here you must configure the proxy port number. The default value is 8080, although this may vary depending on the system administrator settings.

If you are unsure, ask your network administrator or, if your web browser works, copy the settings from there.
For instance, for the firefox web browser, go to Edit->Preferences->Network tab, then click on the Settings... button in the Connection: Configure how Firefox connects to the Internet section.
If the option Manual Proxy Configuration is selected, then you can copy the HTTP Proxy settings from there.
