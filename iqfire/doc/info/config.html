Configuration of the firewall

In this window you can find the options to tune the software to your preferences.

The section is structured in six tabs, where you can:

  • setup options for the generic behaviour of the firewall;
  • personalize the system tray icon behaviour and the popups that it proposes in consequence to some events;
  • personalize some search paths for rules and document loading;
  • personalize the user interface with new styles;
  • manage software updates notifications;
  • configure advanced firewall settings for the root user, such as NAT/Masquerade enable/disable, stateful firewall support, default policy of the packet filter...
