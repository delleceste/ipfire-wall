Input network device

Click on the arrow in the combo box to view a list of available network devices.

The network devices with a plugged network patch are the devices really up and running, while those unplugged are not up at the moment, but they were probably manually added in the past by the user.

If the device you intend to specify is not in the list, you can type its name and add it, if you know that the interface will come up at a certain moment and you want to apply the rule to that interface.
This is for instance the case of the dynamic ppp connections through a modem, which become available after dialing the Internet Service Provider phone number.

The point to point internet connections, such as those provided by a classical Internet Service Provider via a traditional 56k or DSL modem, are usually named ppp and so probably the dynamic network interface created by the linux kernel is something named like ppp0, or ppp1, if you have more than one of such connections.

IP tunnels usually create interfaces named like tun0 or tunX, where X is a number.
