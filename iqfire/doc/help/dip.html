Destination IP address

You can type here the destination address.
iqfirewall supports IPv4 addresses, which are a set of four decimal numbers separated by a dot (.).
Each number must be an integer between 0 and 255. The firewall widgets will definitely help you typing your desired address avoiding mistakes in the syntax. For example, you will not be able to type any character which is not a number or a dot (.), nor will you be able to write a decimal number not in the range 0-255.
