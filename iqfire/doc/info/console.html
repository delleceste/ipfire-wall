The console

This is the iqFirewall console. It shows the packets processed by the firewall, together with their associated response.

  • An accepted packet starts with OK, followed by the number of the rule applied;
  • A blocked packet starts with an X, followed by the number of the rule applied;
  • A line starting with ? represents a packet without a matching rule, to which the firewall default policy is applied.
The information in each line represents:
  1. the packet direction throught the firewall;
  2. the network device involved;
  3. the transport protocol type;
  4. the source ip address;
  5. the source port;
  6. the destination ip address;
  7. the destination port;
  8. the state of the connection;
  9. for the tcp protocol, the flags of the TCP connection;
  10. the name of the rule applied;

Note: given the colors used by the console to represent the packets, it is advised to use the white on black konsole scheme.
