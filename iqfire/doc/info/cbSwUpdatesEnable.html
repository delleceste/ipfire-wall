Notify software updates

iqfire-wall is provided by a system that periodically can check on the Internet if software updates are available. It checks for changes in the release version, that is, for instance, from version 0.98.8 to 0.98.9 or from 0.99.9 to 1.0.

It is recommended that you enable this feature, because with every new release many bugs are fixed and often there are new features or enhancements.

Also remember that, if you want to be sure to have the most up to date software snapshot, you might like to download the source code from the CVS repository of ipfire-wall. Check the online documentation at iqfire-wall download page and visit the sourceforge.net ipfire-wall project page, where you will find more information about using CVS.
