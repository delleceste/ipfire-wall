Log level between user and kernel spaces

This box tunes the level of information that is exchanged between the user and the kernel space, concerning the firewall console. The recommended choice is Smart Log: only unseen packets, because it avoids repeating the identical information many times.

Tune consequently the Time to live of an information entry, in the box above, that establishes the amount of time after which an identical item is shown again on the firewall console.

The issues related to this settings are discussed in this ipfire-wall web page.
