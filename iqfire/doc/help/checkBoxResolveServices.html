Resolve services for the rule match notifier

Check this feature to resove IP addresses and port numbers into the corresponfing internet names and service names.

If your host is not connected to the Internet, disable this feature if the user interface blocks in some situations (while exiting or closing a popup or a window).

Disable this feature also if you are not interested in the internet names and services names or if you do not desire the little system and network overload requested by iqfirewall to accomplish the resolution process.
