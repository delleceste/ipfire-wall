iqFirewall presentation

This is iqfirewall hints section.
In this corner you will find some useful help information about the contents of the firewall window currently displayed.
The behaviour of this element is analogous to the one described for the info section.

  • You can hide/show the help section clicking on the Sidebar Contents toolbar, on the yellow bulb icon;
  • moving your mouse over the window items will then provide useful hints about the usage of each element, for instance, in a line edit used to configure an IP address, this dynamic help will give you some advice on how the IP addresses have to be written.
  • NOTE: the help text is updated with a delay of a few milliseconds after the mouse enters a widget, to avoid continuous changes of the help contents during the user mouse movements over a window. The delay is tunable in the configuration window, under the paths page.
  • Scroll the mouse over an item to scroll the contents of the help or the info text browsers!
  • Usually, bringing the mouse pointer outside the firewall window and then moving it again on the main widget of the window itself, will display the generic help on the section currently shown (this short document is an example).
  • The help and info sections can be vertically resized with the mouse by placing the pointer at the boundary of the two sections (splitter).
  • Also the horizontal space occupied by the help and info panel can be varied with the mouse, by means of a vertical splitter.
