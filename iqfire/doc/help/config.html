Configuration window

The configuration window is made up of a widget divided into tabs each of them representing a subset of the firewall configuration aspects. Click with the mouse on the tab name you want to access and then have a look at all the settings you are intersted in.

As usual in iqfire-wall interface, moving the mouse pointer over each component of the window will display an informational and help message in the info and help panels.

The application of the settigs is a consequence of the Apply button pressing. If, on the other hand, you want to discard the changes, just press the Cancel button to go back to the previous window.
