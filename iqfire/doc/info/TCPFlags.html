tcp specific flags

iqfire-wall lets you configure the tcp flags, which are six fields of the tcp packet that are used to manage, create, synchronize a tcp connection and data flow.
For each tcp flag you will find a check box which must be enabled to specify the requested value for the corresponding bit.

Observing the firewall console, for each tcp packet you will find information about the tcp flags:

  • S for synchronize;
  • A for acknowledge;
  • U for urgent;
  • P for push;
  • R for reset;
  • F for finalize;
