The IGMP protocol

Click this radio button to choose the IGMP protocol for this rule. For this protocol, only the IP options are available.

IGMP is prone to some internet attacks, and firewalls commonly allow the user to disable it if not needed.

Remember that, if you do not need it, you can disable automatically the IGMP protocol simply avoiding to add any rule over it. Actually, iqfire's default policy for packets without any rule provided for them is to drop them. In other words, in iqfirewall, IGMP is disabled by default.
