Add a new stylesheet

Qt can associate a stylesheet to the graphical interface to personalize its appearence. It must be a Cascading Style Sheet (css) document, written following the Qt Style Sheets guidelines from Qt documentation.

As you can see, the table above reports some information about the css document to be loaded:

  • the style name;
  • the Description of the style;
  • the Author;
  • the Date when the style was written;
  • the File Name that is loaded when the row is selected.
