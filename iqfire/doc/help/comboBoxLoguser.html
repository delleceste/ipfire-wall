Log level between the kernel and the user spaces

Click on the combo box and choose the log level that best suits your needs.

It is recommended the Smart Log option, i.e. the first available.
The smart log option optimizes the communication between the kernel and the user space, reducing the redundant information that may result if one of the option below was chosen. Read this web document for an in depth discussion of the issue, or read the info panel explanations over the theme.
