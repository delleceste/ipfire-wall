About iqfire-wall

iqfire-wall manual

iqfire-wall is a Graphical User Interface built using the Qt graphical libraries and based on the ipfire-wall kernel infrastructure.

  • ipfire-wall kernel module;
  • ipfire-wall command line interface and
  • iqfire-wall graphical user interface
have been developed by Giacomo Strangolino, between spring 2005 and summer 2009.

The software was released under the GPL license, a copy of which you should have found in the installation directory of the software (i.e. the directory from which you have executed the installer).
