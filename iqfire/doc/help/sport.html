Source port

You can type here the source port.
A port identifies a service in a host and is an integer positive number less than 65535. The firewall widgets will help you typing your desired port number avoiding mistakes in the syntax. For example, you will not be able to type any character which is not a number, nor will you be able to write a decimal number not in the range 1-65535.

Once you have setup a rule and you have applied it, you can explore the tree widget below this dialog window and see on the information panel a summary of each rule. Moving over a summary, you can click on addresses and port to resolve them into their names, where this is possible.
