How to specify a list of ports

To insert a list of ports proceed like this:

  • type the first port;
  • click on the last apply button on the right. The combo box drop down list should now show the port;
  • click on the + button to add a second element;
  • type in the second port in the white space;
  • click once again on the apply button on the right.
  • Proceed again in the same way until done.
To remove an port from the list:
  • from the drop down list of entries, select the one you want to remove;
  • click on the - button.
