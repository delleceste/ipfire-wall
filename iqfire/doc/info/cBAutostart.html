Start iqfire-wall with Gnome or KDE!

The desktop managers Gnome and KDE, at startup, look inside a particular folder for special files called desktop files. Such kind of files are application launchers and, if placed in that special folder, are run at startup.

iqfire-wall writes an entry in the folders used by Gnome and KDE for autostarting programs, and so this feature should be available with the mentioned desktop environments, together with those managers that follow the freedesktop.org specifications.
