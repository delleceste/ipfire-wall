Obtaining help in installing and using iqfire-wall

The available resources to obtain help to install and configure iqfire-wall are the following:

  • obtain information and hints inside the firewall user interface with the interactive info and help system:
    • enable the help and info widgets by pressing down the icons in the toolbar depicted below: Moving your mouse over the widgets will update the contents of the widgets providing information about them;
    • read the embedded manual by selecting from the menu the item Help -> Firewall Presentation and handbook, or simply pressing the key F1 on the keyboard;
  • read again all the web documentation, starting from the iqfire-wall home and focusing on the problem you have encountered (installatioin, policy management, interpretation of the console messages and so on).
    • In particular, if you run into problems in building or installing iqfire-wall, be sure to have satisfied all the dependencies required by the software. In this case, start reading, or read again, from this page.
  • If none of the methods listed above helps, read and eventually post a message on the ipfire-wall forum hosted at sourceforge.net.
  • To post a message on the forum:
    • login on the web site with your sourceforge username and password. If you haven't got an account, you have to create it on this page;
    • click on the Forums summary link on the menu;
    • choose the help forum to read and post a message on the help threads, or
    • choose the Open discussion forum to read and post on the open discussions about ipfire-wall or the new interface iqfire-wall;
    • click on the thread you are interested in and read or post messages.

Note

The sourceforge.net ipfire-wall project forum is the only place and way to ask help for installing, configuring and running the software.
There one can talk about issues related to the kernel, the ipfire-wall console interface and the graphical user interface named iqfire-wall.
The author will answer only through that channel. Follow the instructions above if you want to visit and post on the forums.
