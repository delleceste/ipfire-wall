Kernel memory usage for dynamic tables

This plot displays kernel memory usage for dynamic tables. Read the information panel for further details.

Displaying details on curve values

Click with the left button of the mouse on a point of the plot near a curve to display a marker with the details of the closest point of the closest curve.

Zooming the plot

Zoom is available on the plot too. Select a region with the mouse left button pressed and then release the mouse button to plot the selected area. Click with the central button to zoom back.
